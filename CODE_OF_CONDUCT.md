# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and keep discussions technical.
Harassment, personal attacks, and discriminatory language are not tolerated
anywhere in the project's spaces (issues, pull requests, discussions).

Violations can be reported privately to the repository owner via GitHub; all
reports will be reviewed and handled confidentially.
